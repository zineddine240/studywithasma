# Next.js Development Skill

When working on Next.js projects, follow these principles:
1. Leverage the App Router or Pages Router according to the project setup.
2. Prefer Server Components by default for better performance, and use Client Components (`"use client"`) only when interactivity or browser APIs are needed.
3. Use Next.js optimized components like `next/image` and `next/link`.
4. Ensure SEO best practices with proper metadata.
5. Follow modular and component-driven architecture for better reusability.
